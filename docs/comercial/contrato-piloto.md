## 1. Objeto

A Contratada fornece acesso ao serviço HealthIntel Core ANS, consistindo em API autenticada com dados públicos da ANS tratados, modelados e disponibilizados para análise de operadoras, beneficiários, mercado e indicadores regulatórios.

## 2. Escopo do Serviço

O serviço inclui:
- Acesso à API Core por 30 dias via 1 (uma) API key
- Endpoints listados na Proposta Comercial anexa
- Documentação técnica (Swagger)
- 1 reunião de onboarding remota

Não estão incluídos dados brutos, exportação integral, TISS real, CNES completo, acesso a camadas bronze/prata/premium ou ambiente dedicado.

## 3. Valor e Pagamento

**Valor total:** R$ 2.500 (dois mil e quinhentos reais)  
**Forma de pagamento:** PIX ou TED antecipado  
**Chave PIX:** [CHAVE PIX]

O acesso é liberado após confirmação do pagamento. Não há reembolso após liberação da API key.

## 4. Uso Permitido

O Contratante poderá utilizar os dados exclusivamente para fins internos de análise, BI e tomada de decisão, vedado:

- redistribuição ou revenda dos dados a terceiros;
- uso em produtos ou serviços comercializados para terceiros;
- scraping automatizado ou tentativa de reconstrução integral da base;
- compartilhamento da API key com terceiros não autorizados;
- qualquer uso que viole a legislação brasileira, incluindo LGPD.

## 5. Confidencialidade

As estruturas, transformações, regras de negócio, enriquecimentos e metodologia de score do HealthIntel Core ANS são de propriedade exclusiva da Contratada e não podem ser divulgados, copiados ou reproduzidos.

## 6. Auditoria e Bloqueio

A Contratada registra logs de consumo por API key. Em caso de uso indevido ou violação deste contrato, a API key será bloqueada imediatamente sem reembolso.

## 7. Disponibilidade e SLA

O serviço é oferecido em modelo best-effort durante o período piloto. Não há garantia de SLA, tempo de resposta ou disponibilidade mínima. Incidentes serão tratados em horário comercial.

## 8. Renovação

Ao final dos 30 dias, o acesso é encerrado. A renovação ou contratação de plano superior deve ser negociada antes do encerramento.
