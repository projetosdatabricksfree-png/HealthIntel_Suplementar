# Glossario

| Termo | Definicao |
|---|---|
| `competencia` | Periodo mensal no formato `YYYYMM`. |
| `trimestre` | Periodo trimestral no formato `YYYYTn`. |
| `registro_ans` | Identificador publico da operadora na ANS. |
| `cd_municipio` | Codigo IBGE de sete digitos do municipio. |
| Gold Analitico | Camada `nucleo_ans` com fatos, dimensoes e marts. |
| Gold Servico | Camadas `api_ans` e `consumo_ans` para entrega ao cliente. |
